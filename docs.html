Overview

Guardify is an AI-powered moderation bot designed for Discord servers.

Violation Categories

The bot monitors various types of content violations. The main categories include:

  • Harassment: Abusive language and behavior.
  • Harassment/Threatening: Aggressive or threatening language.
  • Hate: Hate speech and discriminatory content.
  • Hate/Threatening: Hate speech with threats.
  • Illicit: Content related to illegal activities.
  • Illicit/Violent: Violent content tied to illicit activities.
  • Self-harm: Content related to self-harm or suicidal ideation.
  • Self-harm/Intent: Expressions of intent to self-harm.
  • Self-harm/Instructions: Guidance or instructions to self-harm.
  • Sexual: Explicit sexual content or discussions.
  • Sexual/Minors: Sexual content involving minors.
  • Violence: General violent content.
  • Violence/Graphic: Graphically violent content.

Commands

Guardify provides several slash commands for moderation and configuration:

  • /guard lookup: Lookup a user's moderation logs.
  • /guard strike: Issue a strike to a user, triggering warnings and disciplinary actions.
  • /guard purge: Delete a specified number of messages (up to 100) from a channel.
  • /guard timeout: Temporarily timeout a user for a set duration.
  • /guard panel: Open an interactive settings panel for detailed configuration.
  • /guard am automod setup: Run an automated setup for AutoMod, including role and channel configuration.
  • /guard am scan: Scan recent messages in a channel for violations and delete flagged messages.

/guard panel - Settings Panel

The /guard panel command opens a multi-page interactive settings panel. This panel allows administrators to configure various aspects of the bot’s behavior.

Panel Settings Details (Click to Expand)

Page 1: Automod Settings

  • Toggle AutoMod: Enable or disable the entire AutoMod system.
  • Set Bypass Roles: Specify roles that bypass moderation actions.

Page 2: Message Logging Settings

  • Toggle Message Deletion Logging: Enable or disable logging when messages are deleted.
  • Toggle Message Editing Logging: Enable or disable logging of edited messages.

Page 3: Member Logs Settings

  • Toggle Member Join Logging: Log events when new members join.
  • Toggle Member Leave Logging: Log events when members leave.

Page 4: Violation Categories Configuration

  • Toggle individual violation categories on or off to control which types of content are moderated.

Page 5: Advanced Settings

  • Toggle Auto-Mute: Enable or disable auto-muting when violation thresholds are met.
  • Toggle Use Mute Role: Choose to use a predefined mute role for auto-muting, instead of using the default Discord time-out feature.
  • Set Mute Role: Specify the role that should be applied for muting.
  • Toggle Text Moderation: Enable or disable moderation for textual content.
  • Toggle Image Moderation: Enable or disable moderation for images.
  • Set Warning Message: Customize the warning message sent to users when their content is removed.
  • Set Auto-Mute Duration: Define the duration of the mute when auto-mute is activated.
  • Toggle Log Timeout: Enable or disable logging for timeout actions.
